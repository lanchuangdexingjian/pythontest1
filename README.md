## 功能说明
该脚本根据一组基础用户名/关键词（如 `admin`, `root`, `test` 等），自动生成常见的弱密码变体，包括：
- 大小写变换（全小写、全大写、首字母大写）
- 常见后缀拼接（如 `123`, `!`, `2024`, `01` 等）
- Leet 语替换（如 `a→4`, `e→3`, `i→1`, `o→0`, `s→5`）
- Leet 变体 + 后缀组合

## 输出
生成的密码列表将写入当前目录下的 `wordlist.txt` 文件。

# 端口扫描器（Port Scanner）

这是一个用 Python 编写的简易多线程端口扫描工具，用于快速检测目标主机上指定范围内的开放 TCP 端口。

## 功能简介

- 支持对指定 IP 地址进行端口扫描。
- 可自定义扫描的端口范围（默认为 1–1000）。
- 使用多线程加速扫描过程（最多并发 100 个线程）。
- 超时时间为 0.5 秒，避免长时间等待无响应端口。

## 代码结构说明

### 主要函数

#### `scan_port(ip, port, results)`
- **作用**：尝试连接指定 IP 的某个端口。
- **参数**：
  - `ip`：目标主机 IP 地址（字符串）。
  - `port`：要测试的端口号（整数）。
  - `results`：共享列表，用于记录开放的端口。
- **逻辑**：
  - 创建 TCP 套接字并设置超时。
  - 若连接成功，则将端口加入 `results` 列表。
  - 无论成功与否，最终都会关闭套接字。

#### `scan_ports(target='127.0.0.1', start=1, end=1000)`
- **作用**：启动多线程扫描指定范围内的所有端口。
- **参数**：
  - `target`：目标 IP，默认为本地回环地址 `127.0.0.1`。
  - `start` / `end`：扫描的起止端口号。
- **实现细节**：
  - 每扫描一个端口就启动一个线程。
  - 每当线程数达到 100 时，会等待这批线程完成后再继续，防止资源耗尽。
  - 所有线程结束后，对结果排序并返回开放端口列表。

### 主程序入口

if __name__ == "__main__":
    ports = scan_ports('127.0.0.1', 1, 1000)
    print(ports)
